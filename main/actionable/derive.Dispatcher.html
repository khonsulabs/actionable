Derive Macro actionable::Dispatcher[−][src]

#[derive(Dispatcher)]
{
    // Attributes available to this derive:
    #[dispatcher]
}

Derives the Dispatcher trait.

This trait requires the input parameter to be specified. The full list of parameters that can be customized are:

  • input Type: #[dispatcher(input = "EnumName")]. The enum name here needs to have had Actionable derived on it.
  • Crate name override: #[actionable(actionable = "someothername")]. If you find yourself needing to import actionable as another name, this setting will replace all mentions of actionable with the identifier specified.

The input type must be in scope, as do the derived traits generated by deriving Actionable.
